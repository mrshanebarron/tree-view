# Laravel Design Tree View

Expandable tree view component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/tree-view
```

## Usage

### Livewire Component

```blade
<livewire:ld-tree-view />
```

### Blade Component

```blade
<x-ld-tree-view />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-tree-view-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-tree-view-views
```
